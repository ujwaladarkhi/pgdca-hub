First Semester Subjects

Fundamentals of Computers

Introduction to computer systems, hardware, software and basic operations.

Basic to advanced concepts of C programming language with practical examples.

Code: PGDCA102 Credits: 5

Database Management Systems

Introduction to DBMS, SQL, normalization and database design principles.

Code: PGDCA103 Credits: 4

Second Semester Subjects

Object Oriented Programming with C++

Advanced C++ programming with OOP concepts like classes, inheritance, polymorphism.

Code: PGDCA201 Credits: 5

Web Technologies

HTML, CSS, JavaScript and introduction to front-end web development.

Code: PGDCA202 Credits: 4

Data Structures

Arrays, linked lists, stacks, queues, trees and graph algorithms.

Code: PGDCA203 Credits: 4

Third Semester Subjects

Java Programming

Core Java concepts, multithreading, exception handling and GUI programming.

Code: PGDCA301 Credits: 5

Computer Networks

Network models, protocols, routing, switching and network security basics.

Mobile Application Development

Android app development using Java/Kotlin with practical projects.

Code: PGDCA303 Credits: 4

Cloud Computing

Cloud models, virtualization, AWS/Azure basics and cloud deployment.

Code: PGDCA401 Credits: 4

Data Analytics

Data analysis techniques, visualization tools and basic machine learning.

Project Work

Complete project development covering all phases from planning to deployment.
